# Курсовой проект Никиты Машарина

## Тема: Разработка веб-приложения для подачи заявок на аренду хостинга

### Что готово

 - Конструктор страниц
   - Сборка страниц из шаблонов
 - Взаимодействие с базой данных
   - Вход в БД
   - Отправка запросов и получение выходных данных
     - Булево значение
     - Одно поле
     - Ассоциативный массив
 - Работа с аккаунтами
   - Вход
   - Регистрация
   - Выход
   - Сохранение данных в сессию

### Что в работе

 - Главная страница
 - Локали
 - Работа с хостингами
   - Вывод списка хостингов

### Что в планах

 - Оставшиеся страницы (О нас, Хостинги, Личный кабинет, Подача заявки)
 - Подача заявок
   - Создание новой заявки
   - Истечение срока заявки
 - Админ-панель
