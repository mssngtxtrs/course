# Курсовой проект Никиты Машарина

## Тема: Разработка веб-приложения для подачи заявок на аренду хостинга

### Что готово

 - Конструктор страниц
   - Сборка страниц из шаблонов
 - Взаимодействие с базой данных
   - Вход в БД
   - Отправка запросов и получение выходных данных
     - Булево значение
     - Одно поле
     - Ассоциативный массив
 - Работа с аккаунтами
   - Вход
   - Регистрация
   - Выход
   - Сохранение данных в сессию
 - Локали
   - Подгрузка строк из словарей
 - Вывод списка хостингов
 - Сайт
   - Главная страница

### Что в работе

 - О нас
 - Хостинги
 - Работа с хостингами

### Что в планах

 - Оставшиеся страницы (Личный кабинет, Подача заявки)
 - Локали для оставшихся страниц
 - Подача заявок
   - Создание новой заявки
   - Истечение срока заявки
 - Админ-панель
